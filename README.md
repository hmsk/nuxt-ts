## Setup

- Add `nuxt-ts` dependency using yarn or npm to your project
- Add `nuxt-ts` to `modules` section of `nuxt.config.js`

```js
{
  modules: [
    // Simple usage
    'nuxt-ts',

    // With options
    ['nuxt-ts', { /* module options */ }],
 ]
}
```
